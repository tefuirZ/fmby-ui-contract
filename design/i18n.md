# Design · i18n

## 必备语言

- `zh-CN`（默认）
- `en-US`

可选：zh-TW / ja-JP / ko-KR。

## 实现

- 所有文案外置：`locales/<lang>.json`，键值结构由皮肤自定但需在 manifest 声明
- 用户语言偏好走 [features/settings/appearance.md] 或浏览器 `navigator.language`
- 站点默认语言 `GET /api/site/info` → `language`

## 时区

- 后端时区基线 `Asia/Shanghai`
- 前端展示时间统一格式化（用户 timezone 优先，未设置回退站点 timezone）

## 数字 / 货币

- 数字千分位用 `Intl.NumberFormat`
- 文件大小：1 KB = 1024 B（媒体场景）

## 复数

- 用 `Intl.PluralRules` 或框架内置 plural

## RTL（可选）

- 不强求；如实现需在 manifest 标 `rtl_supported: true`

## 后端文案

- 错误 `error.code`（[api/errors.md]）映射到本地化文案
- `summary` 字段仅作降级 fallback，皮肤优先用 code 翻译
