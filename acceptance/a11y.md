# Acceptance · Accessibility

## 自动化

- axe-core / Pa11y 在 e2e 中跑，0 critical violations
- Lighthouse a11y ≥ 90

## 关键页面手过

- 登录 / 注册
- 首页 / 库 / 详情 / 播放
- 用户管理 / 注册码 / 角色模板
- 媒体资源详情（操作密集）
- pan115 扫码弹窗

## 检查项

- [ ] 所有按钮有可访问名（aria-label 或文本）
- [ ] 表单字段 label 关联
- [ ] 错误关联到字段（aria-describedby）
- [ ] 焦点环可见
- [ ] Tab 顺序符合视觉
- [ ] Modal 焦点陷阱 + Esc 关闭
- [ ] 海报 alt 含标题
- [ ] 图标按钮单独 aria-label
- [ ] 颜色不是唯一信息载体
- [ ] 文本对比度 ≥ AA
- [ ] `prefers-reduced-motion: reduce` 时无大动画
- [ ] 屏幕阅读器通读关键页无阻塞（VoiceOver / NVDA）
